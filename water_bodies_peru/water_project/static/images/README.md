# Imagen DOT.jpg

Coloca aquí la imagen DOT.jpg que será el ícono del chatbot.

La imagen debe ser:
- Formato: JPG o PNG
- Tamaño recomendado: 200x200 píxeles (o proporción 1:1)
- Nombre exacto: DOT.jpg

Esta imagen aparecerá en el botón circular del chatbot en la esquina inferior derecha de la pantalla.

2. INSTALACION RAPIDA (3 PASOS)
================================================================================

PASO 1: Extraer el proyecto
----------------------------
1. Descomprime water_bodies_peru.zip
2. Veras la carpeta: water_bodies_peru\
3. Abre esta carpeta

PASO 2: Configurar API Key
---------------------------
1. Abre el archivo: .env (con Notepad o cualquier editor)
2. Cambia esta linea:
   
   OPENAI_API_KEY=tu-api-key-aqui
   
   Por tu API key real:
   
   OPENAI_API_KEY=sk-proj-ABC123...

3. Guarda el archivo

PASO 3: Ejecutar
-----------------
1. Doble clic en: start.bat
2. Espera 1-2 minutos (instalara paquetes la primera vez)
3. Se abrira automaticamente tu navegador

¬°LISTO! El sistema estara corriendo en http://localhost:8000

4. EJECUCION DEL SISTEMA
================================================================================

Metodo Automatico (Recomendado):
---------------------------------
1. Doble clic en: start.bat
2. El script hara automaticamente:
   - Crear entorno virtual
   - Instalar dependencias
   - Configurar base de datos
   - Iniciar Django (puerto 8000)
   - Iniciar Chainlit (puerto 8001)
   - Abrir navegador

Metodo Manual (Si start.bat falla):
------------------------------------
Abre 2 ventanas de PowerShell/CMD

Ventana 1 - Django:
  cd water_bodies_peru
  python -m venv venv
  venv\Scripts\activate.bat
  pip install -r requirements.txt
  python manage.py migrate
  python manage.py runserver

Ventana 2 - Chainlit:
  cd water_bodies_peru
  venv\Scripts\activate.bat
  cd chainlit_app
  chainlit run app.py --port 8001

Luego abre: http://localhost:8000

Detener el sistema:
-------------------
- Cierra las ventanas de Django y Chainlit
- O presiona Ctrl+C en cada ventana

6. SOLUCION DE PROBLEMAS
================================================================================

Problema: "Python no encontrado"
---------------------------------
Solucion:
- Instala Python desde https://python.org
- Durante instalacion marca "Add Python to PATH"
- Reinicia PowerShell/CMD

Problema: "No se puede activar entorno virtual"
------------------------------------------------
Solucion:
- Abre PowerShell como Administrador
- Ejecuta: Set-ExecutionPolicy RemoteSigned
- Responde: Y (Si)
- Intenta de nuevo

Problema: "Error de OpenAI API"
-------------------------------
Solucion:
- Verifica que .env tenga tu API key correcta
- La key debe empezar con: sk-proj-
- Sin espacios extras
- Reinicia los servidores

Problema: "Template index.html no encontrado"
----------------------------------------------
Solucion:
- Verifica que exista: water_project\templates\index.html
- Si no existe, extrae nuevamente el ZIP completo
- Asegurate de no haber movido archivos

Problema: "Chatbot no aparece"
------------------------------
Solucion:
- Verifica que Chainlit este corriendo en puerto 8001
- Abre en otra pestana: http://localhost:8001
- Si da error, revisa tu API key en .env

Problema: "El mapa no carga datos"
-----------------------------------
Solucion:
- Verifica: water_project\static\data\data.csv
- Abre consola del navegador (F12)
- Busca errores en la pestana Console
